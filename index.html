How do I add videos?
  1. In advance, prepare a folder of your video files. I recommend making them all the same file format and editing the filenames of the movies so they are short and easy to remember and type.
  2. On the main window, click "; videoplayer open"
  3. IN the videobank_player window that opens, click the type of file you wish to load frin the file type dropdown meny(e.g. MPEG = MP4)
  4. drag and drop the folder containing your video files onto the area marked "drop a folder of video files here!
  5. Play videos by typing VIDEO at the start of a chat command
