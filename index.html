ChatMusic User Manual

Installation Instructions

    Install Max MSP from cycling74.com
    The current version of the patch was created on Max 8.2 running on MacOS Ventura (13.0)
    The patch will most likely work on other systems
    You can run Max patches forever with a trial version of Max, but you cannot save changes. The patch is set up so that changes can be saved to external text files, so the patch should work properly with the free version of Max.
    Make a folder in your computer called "ChatMusic".
    Open Max and navigate to Options > File Preferences
    Click the "+" option in the bottom left corner of the File Preferences window to add a path to the file paths of your Max installation.
    Clcik the "choose" button that appears.
    In the menu that appears, navigate to the ChatMusic folder you already created, and press Open.
    Ensure that the "include subfolders" box is checked.
    Now close Max 8.
    Copy the latest version of ChatMusic (which should have a name like ChatMusic_0.2), and the latest version of the externals folder, into the ChatMusic folder.
    If there is an existing version of ChatMusic, you should move it out of the ChatMusic folder if you wish to keep a copy.
    Launch the program by opening "ChatMusic.maxpat" inside the latest version folder. Ensure it opens in Max 8, and not any other version of Max (or a standalone software made in Max which may already be installed on your computer, by rightclicking the file and clicking "Open with", then choosing Max 8)

Setting up the patch

    You should always start by pressing the init button
    if something goes wrong and you need to reset settings, you should first try pressing "lb" and then if that doesn't fix things, try "init" again. If this doesn't work, ensure any custom settings are saved, close the patch, re-open, and make sure to press "init"

FAQ

How does the color coding of patch cords work?

The patch cords are colored to give an indication of the type of information that is carried in them, which can speed up debugging or understanding the logic of the patch.

The cords are coded as follows:

black sends a bang to execute collected data
red pitch information
magenta/pink rhythmic information
cyan/green-blue ASCII code
grey extras attached to notes such as articulation, lyrics
blue messages that clear/empty/reset objects (e.g. zlclear, set, clear)
How do I add videos?
  1. In advance, prepare a folder of your video files. I recommend making them all the same file format and editing the filenames of the movies so they are short and easy to remember and type.
  2. On the main window, click "; videoplayer open"
  3. IN the videobank_player window that opens, click the type of file you wish to load frin the file type dropdown meny(e.g. MPEG = MP4)
  4. drag and drop the folder containing your video files onto the area marked "drop a folder of video files here!
  5. Play videos by typing VIDEO at the start of a chat command
